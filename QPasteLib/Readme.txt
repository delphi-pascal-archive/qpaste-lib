QPasteLib v.3.0 // by Kulikov Viktor (2004)
freeware source from Cool Editor 2005 [cooleditor.nm.ru]

Модуль для подключения поддержки текстовой библиотеки QPasteLib (из невышедшей,
замороженой, версии редактора Cool Editor 2005) с образцами библиотек для текстовых
редакторов.

Пример библиотеки в использовании можно пронаблюдать либо в бете Cool Editor 2005,
либо скомпилировав исходник примера, который прилогается к этой библиотеке.

Вы можете свободно использовать эту библиотеку в своих приложениях, дабы это - проще
простого).

Сам модуль - QPasteLib.pas

Пример использования модуля лежит в папке Demo
[Чтобы запустить демо, скопируйте в папку Demo сам файл QPasteLib.pas и папку libs вместе
со всем ее содержимым].
